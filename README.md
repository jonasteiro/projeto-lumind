## 🛠️ Pré-requisitos
Para rodar este projeto localmente, você precisará de:
* Um servidor local configurado (ex: [XAMPP](https://www.apachefriends.org/), WAMP ou Laragon).
* PHP 8.0 ou superior.
* Servidor MySQL rodando na porta padrão (3306).
* Navegador web moderno.
